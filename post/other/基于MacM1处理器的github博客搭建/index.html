1 hugo下载

https://github.com/gohugoio/hugo/releases

2 hugo安装

解压文件后将hugo移动到制定目录/opt/script/run/hugo,同时配置环境变量export PATH="/opt/script/run:$PATH"

3 测试安装成功：

hugo version

4 建站

hugo new site myblog

5 添加主题

git clone https://github.com/chavinking/hugo-theme-even.git themes/even

6 创建文章

hugo new post/other/基于MacM1处理器的github博客搭建.md

7 启动服务

hugo server –port 9999

9 将文章上传至github

1) 普通上传

hugo cd public git status git add * git commit -m “commit blog” git push -f cd ..

2) 如果将文章上传至已有创库

hugo cd public git init git add * git commit -m “commit blog” git remote add origin https://github.com/chavinking/chavinking.github.io.git git push -f origin master
